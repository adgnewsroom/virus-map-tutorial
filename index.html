Copy of How to update Arkansas covid-19 map

1. Go to this data page:

2. Convert json file to csv

  • That link directs you to a json file. To convert that to a csv file that we can import to datawrapper, go to: arkansasonline.com/json/
  • Copy that long url and paste it into the text box
  • Then hit "DOWNLOAD CSV"
  • You will get a result.csv file

3. Upload your csv to GitHub

  • Go to our GitHub repository, where we store our data: https://github.com/adgnewsroom/virus-tracker
  • Sign in by clicking on the sign in button on the top right corner. The email address is arkdgwebreleases@gmail.com I've shared the password on slack. Ask me if you need it.
  • Click on "Upload files." Choose your result.csv file. Make sure the file name is exactly result.csv otherwise it will not update.
  • Under "Commit changes," put whatever you like in the first textbox. I usually just say update. It's just a line of text helping you keep track of your commits, which isn't that important in our case since all we do is updating our datasets. And then hit "Commit changes."

4. Working with "Missing County Info"

  • The result.csv file has a lot of columns, but just pay attention to the following two, the county name and the number of positive cases. They should be the F and I respectively:
  1. features__properties__county_nam
  1. features__properties__positive
  • Notice the county name cell of the first row says Missing County Info. Assuming that means the state department cannot identify where the cases are, we want to reflect that in our map.
  • Go to the flat page: https://www.arkansasonline.com/admin/flatpages/flatpage/6908/
  • Do a text search on "missing county" You should see two matching results, one for the Arkansas map and one for the national map. Change the number for the Arkansas map if there is a change. The national map usually has thousands of cases without county info.
